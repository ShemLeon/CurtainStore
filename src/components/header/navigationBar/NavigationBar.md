# NavigationBar Component

## Описание

Компонент навигационной панели, отвечающий за отображение главного меню навигации сайта с поддержкой выпадающих подменю.

## Как работает

### Импорты

- Импортирует хук `useDropdown()` для управления состоянием dropdown
- Импортирует данные `navItemsData` с конфигурацией пунктов меню

### Управление состоянием

- Получает из хука функции управления открытием/закрытием dropdown
- Получает функцию проверки состояния открытия dropdown

### Рендеринг

- Рендерит список компонентов `NavItem` для каждого пункта меню
- Передает в каждый `NavItem` необходимые пропсы для управления dropdown
