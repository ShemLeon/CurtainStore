# NavItem Component

## Описание

Компонент отдельного пункта навигационного меню. Отображает ссылку навигации и при необходимости управляет отображением выпадающего подменю (dropdown).

## Пропсы

- **`item`** — объект с данными пункта меню (содержит `name`, `href`, `hasDropdown`)
- **`isDropdownOpen`** — функция для проверки, открыт ли dropdown для данного пункта
- **`onMouseEnter`** — функция-обработчик события наведения мыши
- **`onMouseLeave`** — функция-обработчик события ухода мыши

## Как работает

### Отображение элемента

- Рендерит ссылку (`<a>`) с названием пункта меню из `item.name`
- Отображает иконку стрелки вниз (`fa-chevron-down`), если `item.hasDropdown === true`

### Управление dropdown

- При наведении мыши вызывает `onMouseEnter(item.name)`, если у пункта есть dropdown
- При уходе мыши вызывает `onMouseLeave()` для закрытия dropdown
- Условно рендерит компонент `DropdownMenu`, если:
  - `item.hasDropdown === true`
  - `isDropdownOpen(item.name) === true`

### Стилизация

- При наведении меняет цвет текста на фиолетовый (`#a78bfa`)
- Добавляет верхнюю линию при наведении для визуальной обратной связи
- Использует плавные переходы для всех анимаций
